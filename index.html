學期實作專案成果

Project 0: Two Sum 演算法實作

LeetCode 經典 Python / C++ AI 協同

這是我在 CS351 的起手式專案，其核心任務在於建立一個完整的現代化開發與測試流程。除了演算法的正確性外，更聚焦於利用 AI 工具加速重構並撰寫單元測試。

  • 演算法核心： 實作了時間複雜度 $O(N)$ 的 Hash Map 優化解法。
  • 邊界條件測試： 考慮了重複數字、查無解、以及負數等極端 Edge Cases。
  • 自動化驗證： 撰寫測試腳本，確保任何代碼修改皆不會破壞原有功能。
🔗 專案原始碼
💡 Demo 現場亮點（演算法演進）：

一開始我們先以最直覺的雙層迴圈暴力法（O(N²)）破題，之後透過 AI 的重構提示，改以單一迴圈搭配雜湊表優化，將時間降至 O(N)，這讓我深刻體會到 human-in-the-loop 程式碼校閱的重要性。

Project B: CSV 微型資料庫

系統設計 Data Parsing Error Handling

這是一個功能完整的微型資料庫管理系統。程式能夠精準地對 CSV 檔案進行結構化資料的操作，著重於程式的模組化結構與強健的防呆容錯機制。

  • 核心 CRUD 功能： 支援完整載入（Load）、高清晰顯示（Display）、欄位條件查詢（Search）、新增（Add）及寫回儲存（Save）。
  • 防呆機制： 當遇到 CSV 檔案毀損、路徑錯誤或資料格式不符時，系統能拋出清晰的錯誤提示，而非直接崩潰。
  • 資料架構選擇： 內部使用結構化的高效資料容器，便於隨時擴充欄位。
🔗 專案原始碼
💡 Demo 現場亮點（高難度 Bug 克服）：

在開發初期，我們遇到了包含逗號（,）的資料導致解析錯位的 Bug。我透過主動導引 AI 提供精準的 Regex 規則，成功解決了複雜字串的 Parsing 難題，並加強了資料保存的完整性。
